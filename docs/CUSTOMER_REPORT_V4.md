# Customer-Facing Report V4

The project can generate a customer-facing report that combines:

- business goal
- dataset acceptance
- quality evaluation
- task lifecycle smoke
- release recommendation
- next actions

Run:

```bash
PYTHONPATH=. python scripts/generate_customer_business_report_v4.py
```

Outputs:

```text
outputs/reports_v4/field_service_customer_report.md
outputs/reports_v4/field_service_customer_report.json
```

This is the report you can show to a business stakeholder during pilot review.
